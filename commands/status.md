# /status — Report Current Session Status

When this command is invoked, provide a concise status report:

1. What project am I currently in? (read from project CLAUDE.md or cwd)
2. What tasks have been completed in this session?
3. What task is currently in progress?
4. Are there any pending audit verifications?
5. Are there any blockers?
6. How much of the context window has been used? (estimate based on conversation length)

Keep the report concise — 10 lines maximum.
